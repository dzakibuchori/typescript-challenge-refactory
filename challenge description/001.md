Typescript Challenge

## #1 Palindrome

Create a function to detect [palindrome](https://en.wikipedia.org/wiki/Palindrome).

We already have several palindrome sentences.

```
ibu ratna antar ubi

kasur ini rusak

A nut for a jar of tuna.

Borrow or rob?

Was it a car or a cat I saw?

Yo, banana boy!

UFO tofu?
```
